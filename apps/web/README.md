# Once Web

Frontend application for the Once storytelling platform.

## Tech Stack

Built with Next.js 16 using the App Router. Styled with Tailwind CSS and shadcn/ui components.

## Structure

```
app/            Pages and layouts (App Router)
components/     Reusable UI components
lib/            API client and utilities
hooks/          Custom React hooks
stores/         Client-side state management
```

## Key Pages

The application includes story creation, interactive story sessions, a library of past stories, and a character vault for reusable protagonists.

## Running

```bash
bun run dev
```

Application starts on port 3000.
